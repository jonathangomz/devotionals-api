## Description

An API for retrieve ASD devotionals. \
The database is public for reading and the URL connection can be found in the app module. \
Feel free to use it.

## Installation

```bash
# Clone the repository
$ git clone https://github.com/jonathangomz/devotionals-api.git

# Install the dependencies
$ cd devotionals-api
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
